Personal storage node

Cloud storage from your own machine.

ParaChute is a Go CLI that turns a PC into a self-hosted storage node. Allocate local disk, persist uploads, inspect capacity, and open a React dashboard when you want a visual view.

See Commands Explore Features 
$ parachute setup
$ parachute storage add E:\Storage --limit 500GB
$ parachute server start

Serving dashboard at http://localhost:8080

CLI surface

Manage the node from the terminal.

parachute setup

Initialize config and prepare the node.

parachute storage add <path> --limit 500GB

Allocate a managed storage root on local disk.

parachute storage list

View configured roots and capacity limits.

parachute server start

Run the API and local dashboard.

Features

Small core, useful building blocks.

01

Persistent uploads

Files are streamed into object storage and paired with JSON metadata.

02

Storage roots

Allocate one or more local folders or drives with explicit capacity limits.

03

Node insights

Inspect allocated capacity, used space, upload count, and utilization charts.

04

Secure path

Optional WireGuard setup provides a foundation for private remote access.

Dashboard

A visual admin surface when the CLI is not enough.

The React dashboard focuses first on node insights: total allocated storage, used and free differential, roots, uploads, and utilization trends. Rig stats like physical drives and platform details live on a separate page.
