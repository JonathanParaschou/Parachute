What it does

A lightweight node for storage you control.

01

Persistent Uploads

Files are written into managed storage roots with object files and JSON metadata sidecars.

02

Node Insights

Track allocated capacity, used and free space, root count, upload count, and utilization trends.

03

Local First

Run the node on Windows, macOS, or Linux using a small Go backend and a React dashboard.

04

Secure Access Path

Optional WireGuard setup gives the project a path toward private remote access.

Dashboard

Node visibility first, machine details second.

The app opens to storage intelligence: total allocated storage, used and free differential, storage roots, upload count, utilization charts, upload volume, and cumulative stored data. Hardware details live on a separate Rig Stats view.

Quickstart

Build, allocate storage, start the node.

git clone <repository-url>
cd parachute
go build ./cmd/parachute

parachute setup
parachute storage add /path/to/drive --limit 500GB
parachute server start
